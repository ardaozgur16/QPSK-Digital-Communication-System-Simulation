# QPSK-Digital-Communication-System-Simulation
Digital Communication System Simulation (QPSK) — Python, NumPy   Simulated an end-to-end digital communication chain using QPSK (Quadrature Phase Shift Keying) modulation.   Modeled AWGN channels and implemented algorithms for symbol mapping, demodulation, and Bit Error Rate (BER) calculation to evaluate transmission performance.
